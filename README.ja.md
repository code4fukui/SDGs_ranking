# SDGs_ranking

このリポジトリは、都道府県のSDGsに関する認知度やSDGs施策の実施状況を可視化するためのアプリケーションです。

## デモ
https://code4fukui.github.io/SDGs_ranking/

## 機能
- 都道府県のSDGsに関する認知度やSDGs施策の実施状況をランキング形式で表示
- 各都道府県のスコアをカラー表示した日本地図を表示
- 都道府県をクリックすると、その地域の関連情報が表示される

## データ・API
- データソース: [ビジネスパーソンに聞く都道府県「SDGs認知度」ランキング＜2021＞｜新・公民連携最前線｜PPPまちづくり](https://project.nikkeibp.co.jp/atclppp/PPP/report/090200284/)
- スクレイピングツール: [scrapeCSV.js](https://js.sabae.cc/scrapeCSV.js)
