# SDGs_ranking

このリポジトリは、都道府県のSDGsに関する認知度やSDGs施策の実施状況を可視化するウェブアプリケーションです。

## デモ
https://code4fukui.github.io/SDGs_ranking/

## 機能
- 都道府県のSDGs認知度スコアとSDGs施策認知度スコアを日本地図上に表示
- 都道府県をクリックすると、その地域の関連情報が表示される
- 表示指標を切り替えられる

## データ・API
本アプリケーションは、以下のオンラインレポートからスクレイピングしたデータを使用しています:
- [ビジネスパーソンに聞く都道府県「SDGs認知度」ランキング＜2021＞｜PPPまちづくり](https://project.nikkeibp.co.jp/atclppp/PPP/report/090200284/)
